# ToDoManagerPro

Керування задачами — фінальний проєкт курсу з React.

## Опис

ToDoManagerPro — простий веб-додаток для керування списком задач.  
Користувач може:

- додавати, редагувати та видаляти задачі
- відзначати зроблені
- фільтрувати (всі / активні / завершені)
- створювати, редагувати та видаляти категорії для задач

## Демонстрація

Живий приклад доступний тут:  
[https://to-do-manager-pro.vercel.app](https://to-do-manager-pro.vercel.app)

## Стек технологій

- React (функціональні компоненти, хуки)
- Vite — сучасний інструмент збірки
- HTML, CSS (теми та стилі)
- Локальне зберігання задач (localStorage)
